# NationGraph Sales Workspace Demo 

built with Next.js.

## Structure

- `app/`: main application

## Features

- Opportunity workspace UI
- Stakeholder graph with React Flow
- Demo AI copilot powered by Backboard
- Demo feedback loop and outreach drafting

## Run

```bash
npm install
cp .env.example .env
npm run dev
```

## Checks

```bash
npm test
npm run lint
npm run build
```
